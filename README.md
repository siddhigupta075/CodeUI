# CodeUI.apk – Design Overview

This project showcases a modern Android UI built using **Jetpack Compose**, inspired by a cosmic and glassmorphism design style. The interface uses a dark theme with soft gradients, translucent surfaces, and subtle white borders to create depth and visual clarity.

Key elements include reusable glass components, smooth animations, and a rotating central object to establish visual hierarchy and engagement. Typography is enhanced with shadows to ensure readability over complex backgrounds.

The layout focuses on clean spacing, intuitive navigation, and consistency across screens. Overall, the design aims to deliver a minimal, immersive, and aesthetically pleasing user experience.
